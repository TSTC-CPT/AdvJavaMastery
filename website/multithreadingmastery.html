Threading Mastery Assignment

Optimize your program by executing the queries in your program on a separate thread. When loading the data for the lodges, query the database for all of the information, including images, in a single query. Execute that query on its own thread. Update your application’s interface as each row from the result set is read in. You should be able to move about the program freely as the images and other data load in real time.

Add HTML reports to your program. Use CSS to add color and style to your reports. Customer reports will show order history for a given time frame. Employee reports will list the customers and their total spending for a given time frame. Manager reports will list total revenue for the lodges over a given time frame. All time frames will be selected with a GUI calendar. Open the report in the default browser and save a copy of the HTML file in a local folder called “Reports”. Store the Reports folder in the user's Documents folder (Windows OS). Generate the reports on a separate thread.

You can use this code to get the path to the Documents folder: FileSystemView.getFileSystemView().getDefaultDirectory().getPath();

Submission notes: The project must be turned in as a zipped Netbeans (Ant) project.
Your name must be part of the project name, i.e. JohnDoeMastery
Briefly comment your code and include a heading in your main file to specify the module number.
Classes should be developed in separate files. Code must use the naming convention specified by the course instructor. Also be sure to follow all rules listed in the course and in the syllabus.
