How to install Netbeans and the JDK

All projects in this course must be turned in as zipped Netbeans projects.

Download version 17 of the JDK.opens in new window Version 17 is Oracle’s current long term support version.

Oracle may require you to create an account. Run the installation program and complete the installation.

After installing, add the JDK 17 bin folder to the system path. To do this click "start" and type “path”.

Click on, "Edit the system environment variables".

Click the button that is labeled "Environment variables"

Double click "Path" from the lower "System variables" window.

Click "new", then browse to the JDK bin folder.

After adding it to the path, click "Move Up" until that line is at the top.

Download and install Netbeansopens in new window

Create your Java projects using Ant, do not use Maven

Give your project a name that includes your name and a directory where you can find it.

 Example: johnDoeLab1

The first time you create a project Netbeans may ask you to activate Java. Click "download and activate".
