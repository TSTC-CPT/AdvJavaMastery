Inheritance Lab

Create a base class called Vehicle. Add the following members:

  • year
  • make
  • model
  • display()

Derive subclasses called Automobile and Motorcycle with the following members:

Automobile:

  • number of seats
  • hasManualTransmission
  • towingCapacity
  • display()

Motorcycle:

  • wheelCount
  • engineSize
  • display()

Derive 2 more classes from each of these 2 subclasses. You will need to think of 2 specific Automobiles and 2 specific motorcycles and create classes for them. This program will then have a total of 7 classes. Each class must have at least one constructor.

These last 4 classes only need to contain a constructor and no other methods. Use these constructors to set all of the member variables.

In main() create an array of 4 Vehicles. Assign each element of the array an object from each of your 4 lowest level subclasses. Create the objects using constructors. Use super to call the base class constructor. Use a for loop to call display() on all 4 objects.

The display() methods will show the labeled values of the class’s member variables. You may print directly from the method, or return a String to main()
