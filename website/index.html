Inheritance Mastery Assessment v1.3

In this program you will begin creating classes for the Punching Parrot Travel Agency beginning with a lodging booking service. A lodge (or lodging) is the general term for a place to stay. Lodges may be a home such as an Air B&B or a standard hotel. In this assignment you will be developing some classes for the various objects that will be used in the travel agency’s program. You will then need to test the classes by writing a console program to work with the classes. Objects will be created from the classes. These objects can then be used in the program in accordance with user input.

Develop the classes listed in the bullet points below. Determine which classes should take advantage of inheritance and derive them accordingly. Each class should have at least 2 constructors. The first is the default constructor and the second is defined to your liking. All member variables must be defined in the constructors. Subclass constructors must invoke the base class constructor. Remember that anytime inheritance is used you should be able to say: <child class name> is a <parent class name>. Example: Orange is a Fruit. If this statement doesn’t make sense, then inheritance should not be used in this way. Incorrect Example: Hero is an Inventory. Neither Hero or Inventory would be derived from one another.
