MySQL Mastery Assignment

For this assignment you will be setting up tables in your own MySQL database. Use the schema and login information provided by your instructor. Information in your program will now need to be saved and retrieved from the remote database, with the local files serving as a backup. This includes all account information, lodges, and images.

Add images for the lodges in the program. From the Customer View each lodge will have several images (3 minimal) reduced to the size of a thumbnail. If a user clicks on an image a larger viewing area will open that will allow the user to cycle through the images. Add an ArrayList to your Lodge class to store the images for each lodge.

The employee view will allow the user to add images to a lodge. Use the JFilechooseropens in new window to select image files and allow the employee to select multiple image files at once. The images will be stored in the database. You may want to create a separate table for the images or chain the byte arrays together as a single blob. Using a separate table is probably easier. Make sure each lodge can support as many images as the employee wants to add.

The Manager view will not incorporate any new functionality just make the current functions work with the database.

Add HTML reports to your program. Use CSS to add color and style to your reports. Customer reports will show order history for a selectable time frame. Employee reports will list the customers and their total spending for a selectable date range. Manager reports will list total revenue for the lodges over a selectable date range. All time frames will be selected with a GUI calendar or date selector comboboxs. Open the report in the default browser and save a copy of the HTML file in a local folder called “Reports”. Store the Reports folder in the user's Documents folder (Windows OS).

Optimize your program by reducing the number of calls to the database to as few as possible.

Submission notes: The project must be turned in as a zipped Netbeans (Ant) project.
Your name must be part of the project name, i.e. JohnDoeMastery
Briefly comment your code and include a heading in your main file to specify the module number.
Classes should be developed in separate files. Code must use the naming convention specified by the course instructor. Also be sure to follow all rules listed in the course and in the syllabus.
