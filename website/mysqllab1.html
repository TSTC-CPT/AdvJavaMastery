MySQL Lab 1

This program will allow a user to log into and access a remote database. The program will work with a single table and the user will be able to insert information into that table. They will also be able to view all of the information in the table.

Begin with a small login window that prompts for a name and password. Add a tooltip to the name and password labels that reveals the correct name and password for you database. If the user enters the wrong credentials clear the password field.

If the connection is successful expand the Jframe or close the original and open a larger one. The program will access a single table in the database called People. You will need to create this table in your schema and populate it with at least 3 entries. This table should have columns for name and age. On the left side of the window provide 2 labeled text fields, 1 for name and 1 for age. On the right side show all of the data in the table in rows and columns. You may use Labels, a JTextArea, JTextPane or JTable (shown) to display the information.

There will be 2 buttons on the second window: “Add” and “Close”. Clicking Add will verify the name and age fields contain valid information. If the name and age are valid:

  • Add them to the database
  • Refresh the window with the new data that is being read from the database
  • Clear the name and age fields

If the information is not valid indicate which is not correct by changing the background color of that field to light red or dull pink.
